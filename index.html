How to Use the Calculator

1. Use the number buttons (0-9) to input your numbers.

2. Use the operator buttons (+, -, *, /) to perform calculations.

3. Press "AC" to clear the input.

4. Press "DE" to delete the last character in the input.

5. Press "=" to see the result of your calculation.

6. You can use "." to enter decimal numbers.

7. Make sure to use operators for multiple-step calculations.
